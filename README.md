# Hospital Management System (HMS) 🏥

A robust and efficient **C++ Console Application** designed to streamline hospital administration. This project focuses on managing patient records and doctor schedules using professional software engineering practices.

##  Core Concepts (OOP)
This project is built from the ground up using **Object-Oriented Programming (OOP)** to ensure the code is clean, modular, and scalable. Key concepts applied:
*   **Classes & Objects:** Modeled real-world entities like `Patient` and `Doctor`.
*   **Encapsulation:** Protected sensitive data through private members and public getters/setters.
*   **Inheritance:** Used a base class `person` for shared attributes between patients and doctors.

##  Features
*   **Patient Management:** Add, search, edit, and delete patient information.
*   **Doctor Directory:** Manage medical staff records and specialization.
*   **User-Friendly Menu:** Interactive console-based UI for easy navigation.
